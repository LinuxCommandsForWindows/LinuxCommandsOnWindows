A series of commands ported to Windows because why not?

| ------------------------------------------------------------------------------------------------------ | --------------------------------------------------- |
| [`whoami`](https://github.com/LinuxCommandsForWindows/LinuxCommandsOnWindows/tree/main/src/whoami)     | `whoami` ported to Windows.                         |
